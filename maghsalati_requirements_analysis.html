لوحة الإدارة — Admin Dashboard (React.js)

إدارة الاشتراكات أولوية عالية

  • خطط اشتراك للمغاسل (شهري / سنوي)
  • نظام خصومات وأكواد ترويجية (Promo Codes)
  • إضافة / تعديل / حذف خطط الاشتراك
  • متابعة حالة اشتراك كل مغسلة
  • تجديد تلقائي أو يدوي للاشتراكات

إدارة المستخدمين والمغاسل أولوية عالية

  • عرض ملفات جميع المستخدمين
  • تفعيل / تعليق / حظر الحسابات
  • إضافة / تعديل / حذف أي حساب
  • تفعيل الجهاز البديل عند تغيير الهاتف
  • إدارة فترات التجربة المجانية

لوحة المؤشرات التحليلية أولوية عالية

  • مراقبة إيرادات اشتراكات المغاسل
  • حركة العملاء وأنماط الاستخدام
  • المغاسل الأكثر نشاطاً
  • تقارير Python للتحليل المتعمق
  • رسوم بيانية وإحصاءات مرئية

إدارة الإعلانات أولوية متوسطة

  • رفع بانرات للمستخدمين أو المغاسل
  • مكونات الإعلان: عنوان + صورة + نص
  • جدولة الإعلانات وتحديد الجمهور
  • إضافة / تعديل / حذف الإعلانات

الدعم الفني والشكاوى (Ticketing) أولوية متوسطة

  • استقبال شكاوى العملاء والمغاسل عبر الرسائل المباشرة
  • تتبع حالة كل تذكرة دعم (مفتوحة / قيد المعالجة / مغلقة)
  • الرد المباشر على المستخدمين من الداشبورد
  • تعديل أرقام التواصل الظاهرة للمستخدمين
  • دعم الاتصال الهاتفي وربطه بالنظام
تطبيق العميل — Customer App (React Native)

إدارة الحساب أولوية عالية

  • التسجيل بالاسم ورقم الهاتف + OTP
  • معرف فريد (Unique ID) لكل عميل
  • QR Code قابل للمسح بكاميرا المغسلة
  • صورة شخصية اختيارية
  • تسجيل دخول برقم الهاتف + رمز SMS

الاستكشاف والفلترة أولوية عالية

  • خريطة للمغاسل القريبة (Maps API)
  • فلترة: الأعلى تقييماً
  • فلترة: الأرخص سعراً
  • فلترة: الأقرب موقعاً
  • عرض تفاصيل كل مغسلة (ساعات، سعر)

تتبع الطلبات (Tracking) أولوية عالية

  • عرض حالة الملابس لحظة بلحظة
  • إشعار فوري عند كل تغيير في الحالة
  • مراحل: استلام → غسيل → كوي → جاهز
  • تاريخ وتوقيت كل مرحلة

التواصل والتقييم أولوية متوسطة

  • محادثة نصية (Chat) مع المغسلة
  • تقييم ذكي: يُفتح فقط عند اكتمال الفاتورة
  • منع التقييمات الوهمية بالربط بالفاتورة
  • نظام نجوم + تعليق نصي

سجل الفواتير أولوية متوسطة

  • عرض جميع الفواتير مع التفاصيل الكاملة
  • تتبع الديون الآجلة (إن وجدت)
  • تصفية حسب التاريخ أو الحالة
  • إمكانية تحميل أو مشاركة الفاتورة

العروض والإعلانات أولوية منخفضة

  • شاشة مخصصة لعروض المغاسل
  • إعلانات عامة من الإدارة
  • إشعارات عروض خاصة
تطبيق المغسلة — Laundry App / POS Panel

إدارة الملف التجاري أولوية عالية

  • اسم المحل ورقم التواصل للعملاء
  • الموقع الجغرافي الدقيق
  • أوقات العمل (ساعات الدوام)
  • صورة البروفايل (اختيارية)
  • رقم الإدارة الداخلي

دليل الأصناف (Menu) أولوية عالية

  • إنشاء أقسام (رجالي / نسائي / أطفال...)
  • إضافة أصناف داخل كل قسم + السعر
  • مثال: رجالي → ثوب كوي → 3 ريال
  • تعديل وحذف الأصناف والأقسام
  • البحث السريع عن الصنف

نظام الفواتير (POS) أولوية عالية

  • إنشاء فاتورة بمعرف العميل أو QR
  • إضافة أصناف وحساب المجموع تلقائياً
  • أنواع الدفع: نقدي / شبكة / آجل
  • تغيير حالة الفاتورة مع إشعار للعميل
  • تعديل الفاتورة مع إشعار تلقائي

طباعة ومشاركة الفواتير أولوية عالية

  • طباعة بلوتوث (طابعة حرارية)
  • طباعة WiFi
  • تصدير PDF
  • إرسال للعميل عبر WhatsApp
  • إشعار داخل التطبيق

إدارة العملاء (CRM) أولوية متوسطة

  • سجل العملاء المتعاملين مع المغسلة
  • عرض جميع فواتير كل عميل
  • تتبع مبالغ الآجل المتأخرة
  • تنبيه تلقائي للعميل عند تعديل فاتورته
  • إرسال تذكير يدوي للديون

الترويج والتقارير أولوية متوسطة

  • إنشاء عروض خاصة بالمغسلة
  • تقرير يومي: نقدي + آجل + إجمالي
  • ملخص الدخل حسب نوع الدفع
  • تقارير أسبوعية وشهرية

تحليل المخاطر والتحديات
عاليتكلفة SMS/OTP مرتفعة عند التوسع — يُنصح بالتفاوض مع مزود خليجي محلي
عاليتوافق الطباعة مع طابعات بلوتوث المتنوعة — اختبار مبكر مع أكثر 3 موديلات شيوعاً
عاليأداء خرائط المناطق الريفية في اليمن — وضع خطة بديلة لإدخال الموقع يدوياً
متوسطتعقيد State Machine — أي خلل يؤثر على الفواتير والتقييمات مباشرة، يجب اختباره بدقة
متوسطتوافق WhatsApp Business API — بعض الدول قد تواجه قيوداً، يلزم وجود بديل (PDF)
متوسطتبني المغاسل للتكنولوجيا — كثير منها لا تزال تستخدم الورق، يلزم واجهة بسيطة جداً
منخفضالمنافسة المحلية — السوق لا يزال في بداياته وغير مشبع بحلول متكاملة للمغاسل
منخفضالتوسع إلى دول جديدة — المنصة SaaS مرنة للتوسع بإضافة عملات ومناطق جديدة
